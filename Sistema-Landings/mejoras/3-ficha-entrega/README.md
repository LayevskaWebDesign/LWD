# Mejora 3 · Ficha de entrega

**Para qué:** convertir el envío de las versiones en **un solo paso de venta**. La ficha le
presenta al cliente las versiones generadas, qué incluye cada plan, el precio y **la recomendación
del estudio** — diseñada para sugerir el upsell (Basic → Animation → Super Pro).

## Cómo usarla
1. Copiá [`ficha-entrega.plantilla.md`](ficha-entrega.plantilla.md) a la carpeta del cliente como
   `clientes/<cliente>/ficha-entrega.md`.
2. Llená los `{{campos}}`: nombre, links de preview (Netlify o localhost), recomendación.
3. Enviásela al cliente como PDF o link. (Si querés PDF, se puede exportar el Markdown.)

## Recomendación de venta
- Mostrá **siempre las versiones lado a lado** (los 3 links). Ver el Super Pro junto al Animation
  es lo que vende el Super Pro.
- En "Recomendación del estudio" empujá al **Animation como mínimo**, y posicioná el Super Pro
  como la opción premium para quien quiere destacar de verdad.

> Activación por prompt: `MEJORAS OPCIONALES: ficha` (Claude la genera ya prellenada con el contexto).
