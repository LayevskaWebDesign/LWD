# Paleta y tipografía — RODMA

> HEX **aproximados** leídos del logotipo (eyeball). Si tenés los valores exactos de marca,
> reemplazalos. Igual conviene afinarlos al generar para garantizar contraste WCAG AA.

## Colores (HEX)
| Rol | HEX (aprox.) | Nota |
|---|---|---|
| Primario | `#1B2C4B` | azul marino del logo — fondos, texto fuerte, header |
| Acento / CTA | `#E84C3D` | rojo coral del logo — botones, llamados a la acción |
| Secundario | `#C9A9DD` | lila (fondo de algunas piezas) — acento suave, opcional |
| Fondo claro | `#F7F5FA` | base clara |
| Blanco | `#FFFFFF` | superficies, texto sobre navy/coral |
| Texto principal | `#16233F` | navy casi negro |
| Texto tenue | `#5B6472` | secundario |

> Contraste: usar **navy** para texto largo; el **coral** va bien en botones grandes y títulos
> (con texto blanco, revisar tamaño para cumplir AA). El lila solo como acento, no para texto.

## Tipografía
- **Titulares:** serif robusta / slab tipo el logotipo RODMA (a elegir equivalente web: ej. *Arvo*, *Roboto Slab*, *Zilla Slab*).
- **Cuerpo:** sans-serif limpia y legible (ej. *Inter*, *Manrope*) — a confirmar.
- **Íconos:** Lucide React (por defecto del estudio).

## Notas de estilo
Bold, enérgico y amigable. Mucho contraste navy/coral, aire generoso, esquinas redondeadas suaves.
El círculo navy del logo sirve de **favicon/avatar**.
